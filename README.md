# EDMs PDF Parser

A simple(ish) parser for converting PDF-formatted EDMs documents into HTML5

## Prerequisites

- [Pdftotext](http://www.foolabs.com/xpdf/download.html)
- [Kindlegen](http://www.amazon.com/gp/feature.html?ie=UTF8&docId=1000234621)
